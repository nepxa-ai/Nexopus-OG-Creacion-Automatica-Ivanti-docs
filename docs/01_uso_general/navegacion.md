# Navegación general

La interfaz de **Nexopus** está diseñada para ser intuitiva y modular.

## Elementos principales

- **Barra lateral izquierda:** contiene los módulos y accesos directos.
- **Encabezado superior:** muestra tu nombre, perfil y el buscador global.
- **Área central:** muestra las vistas, formularios y reportes.
- **Notificaciones:** se despliegan en la esquina superior derecha.

---

!!! note "Consejo"
    Si no encuentras un módulo, verifica tus **permisos** o comunícate con el administrador.
